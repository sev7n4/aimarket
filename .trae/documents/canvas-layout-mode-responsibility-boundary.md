# 画布布局模式职责边界与业务流转设计

## 设计概述

本文档定义了纵向滚动模式和自由布局（精修）模式的职责边界，以及用户在两种模式间的业务流转流程。

---

## 核心定位

### 纵向滚动模式
- **定位**：历史浏览模式
- **用户意图**：查看生成历史、对比批次结果、选择图片进行下一步操作
- **交互方式**：自然滚动、点击选中
- **适用场景**：日常使用、移动端、所有用户

### 自由布局模式（精修模式）
- **定位**：单图精修模式
- **用户意图**：对选中的单张图片进行深度编辑，产出工业级成品
- **交互方式**：拖拽、缩放、平移、AI工具链联动
- **适用场景**：专业编辑、PC端、需要精细调整的场景

---

## 业务流转设计

### 流转流程图

```
纵向滚动模式（查看）
       │
       │ 点击图片 → 显示操作栏
       │
       ├─────────────────────────────────────┐
       │                                     │
       ↓                                     ↓
  [预览] → Lightbox                   [精修] → 自由布局模式
       │                                     │
       │ 关闭 ←─────────────────────────────┘
       │                                     │
       ↓                                     ↓
  返回纵向模式                    精修完成 → 自动返回纵向模式
                                            │
                                            ↓
                                    聚焦该批次查看结果
```

### 流转触发点

| 触发方式 | 行为 | 说明 |
|---------|------|------|
| 点击图片上的"精修"按钮 | 进入自由布局，聚焦该图 | 主要入口 |
| 点击批次下方的AI工具 | 进入自由布局，聚焦该图 | 快捷入口 |
| 点击工具栏的"自由布局"按钮 | 进入自由布局，显示所有图片 | 用于对比多图 |
| 精修完成后点击"返回" | 返回纵向滚动，聚焦该批次 | 明确退出 |
| 快捷键 Ctrl+L | 切换模式 | 高级用户入口 |

## 图片选中操作栏设计

### 操作按钮

选中图片后显示浮动操作栏，包含三个按钮：

| 按钮 | 图标 | 功能 | 说明 |
|------|------|------|------|
| **预览** | Eye | 打开 Lightbox | 双击图片也可触发 |
| **精修** | Wand2 | 进入自由布局模式 | 自动聚焦该图，隐藏其他图片 |
| **删除** | Trash2 | 删除该图 | 确认后删除 |

### 不保留的按钮

- **下载**：通过右键菜单或选中后点击工具栏下载按钮完成
- **上传**：纵向模式下不需要

## 纵向滚动模式设计

### 功能支持

| 功能 | 是否支持 | 说明 |
|------|---------|------|
| 查看历史批次 | ✅ 核心功能 | 自然上下滚动浏览 |
| 选中图片 | ✅ | 点击选中，显示操作栏 |
| 预览放大 | ✅ | 双击或点击"预览"按钮 |
| 精修入口 | ✅ | 点击"精修"按钮进入自由布局 |
| 删除图片 | ✅ | 点击"删除"按钮 |
| AI 工具入口 | ✅ | 批次下方快捷入口 |
| 画布工具栏 | ❌ | 不需要 pan/zoom/grid |
| 拖拽移动 | ❌ | 不支持，保持系统布局 |
| 缩放画布 | ❌ | 不支持，使用固定网格 |
| 撤销/重做 | ⚠️ 有限 | 仅支持删除撤销 |

### UI布局

```
┌─────────────────────────────────────┐
│ [纵向滚动]                    ← 模式指示器
├─────────────────────────────────────┤
│ 批次 1 · 简约产品图                  │
│ 05-29 14:30 · 4 张结果              │
├─────────────────────────────────────┤
│ ┌───┐ ┌───┐ ┌───┐ ┌───┐            │
│ │主图│ │卖点│ │场景│ │详情│ ← 网格展示 │
│ └───┘ └───┘ └───┘ └───┘            │
│   ↓ 选中后显示                       │
│   [预览] [精修] [删除]              │
├─────────────────────────────────────┤
│ [变体] [扩图] [裁剪] [擦除] [编辑]   │ ← AI工具链入口
└─────────────────────────────────────┘
```

### AI工具链点击行为

| 工具 | 点击后行为 | 说明 |
|------|-----------|------|
| 变体 | 进入精修模式 | 需要选图进行变体生成 |
| 扩图 | 进入精修模式 | 需要选图进行扩图 |
| 裁剪 | 进入精修模式 | 需要选图进行裁剪 |
| 擦除 | 进入精修模式 | 需要选图进行擦除 |
| 编辑 | 保持纵向模式 | 在工作台修改prompt重新生成 |

## 自由布局（精修）模式设计

### 功能支持

| 功能 | 是否支持 | 说明 |
|------|---------|------|
| 查看历史批次 | ⚠️ 有限 | 其他图片淡出或缩小到角落 |
| 选中图片 | ✅ | 已选中目标图片 |
| 预览放大 | ✅ | 双击打开 Lightbox |
| 焦点编辑 | ✅ | 点击图片添加焦点 |
| 拖拽移动 | ✅ 核心功能 | 自由调整图片位置 |
| 缩放画布 | ✅ 核心功能 | 查看细节或全局概览 |
| 撤销/重做 | ✅ 完整 | 支持位置/删除撤销 |
| Brush 编辑 | ✅ | 圈选区域编辑 |
| AI 工具链 | ✅ | 工具栏下方 |
| 返回纵向模式 | ✅ | 明确的返回按钮 |

### UI布局

```
┌─────────────────────────────────────┐
│ [← 返回]  精修：批次1-主图           │ ← 返回按钮 + 标题
├─────────────────────────────────────┤
│                                     │
│         ┌─────────────┐             │
│         │   选中的图   │  ← 居中放大 │
│         │             │             │
│         └─────────────┘             │
│                                     │
│  [其他图片缩略图] ← 可选显示         │
├─────────────────────────────────────┤
│ [选择] [移动] [缩放] [撤销] [重做]   │ ← 画布工具栏
│ [变体] [扩图] [裁剪] [擦除] [编辑]   │ ← AI工具链
└─────────────────────────────────────┘
```

### 精修完成后的行为

1. 生成的图片追加到当前批次
2. 自动返回纵向滚动模式
3. 聚焦该批次，显示新生成的图片

## 移动端特殊处理

### 纵向模式

| 功能 | 支持 | 说明 |
|------|------|------|
| 滑动浏览 | ✅ | 原生滚动 |
| 点击选中 | ✅ | 显示浮动操作栏 |
| 预览 | ✅ | 点击按钮或双击 |
| 精修 | ✅ | 进入单图精修模式 |
| 删除 | ✅ | 点击按钮 |
| 长按菜单 | ✅ | 下载等操作 |
| AI工具链 | ✅ | 批次下方 |

### 精修模式

| 功能 | 支持 | 说明 |
|------|------|------|
| 双指缩放 | ✅ | 缩放画布 |
| 单指拖动 | ✅ | 移动图片 |
| Brush圈选 | ✅ | 手指圈选 |

- **不提供自由布局切换按钮**：避免用户误操作

- **精修模式强制单图**：避免复杂操作

## 实现要点

### 代码层面

1. **纵向模式简化**
   - 移除画布工具栏（pan/zoom/grid）
   - 保留 AI 工具链入口
   - 添加图片选中操作栏组件

2. **精修模式增强**
   - 进入时自动 `fitToItem` 聚焦目标图片
   - 其他图片淡出或缩小到角落
   - 添加明确的"返回"按钮
   - 完成后自动返回纵向模式

3. **状态传递**
   - 精修目标图片 ID 在模式切换时传递
   - 选中状态在两种模式间共享

### 组件结构

```
DesignCanvas
├── ScrollModeCanvas（纵向滚动）
│   ├── BatchSection（批次区域）
│   │   ├── ImageGrid（图片网格）
│   │   ├── ImageItem（图片项）
│   │   │   └── ImageActionBar（操作栏：预览/精修/删除）
│   │   └── AiToolChain（AI工具链）
│   └── ScrollToolbar（简化工具栏）
│
├── FreeModeCanvas（自由布局/精修）
│   ├── TargetImage（目标图片，居中放大）
│   ├── OtherImages（其他图片，淡出或缩略）
│   ├── CanvasToolbar（完整画布工具栏）
│   ├── AiToolChain（AI工具链）
│   └── ReturnButton（返回按钮）
│
└── CanvasLightbox（预览组件）
```

## 总结

### 核心设计原则

1. **纵向滚动 = 查看 + 选择**：用户浏览历史、选择图片
2. **自由布局 = 精修 + 创作**：用户对单图深度编辑
3. **业务流转清晰**：查看 → 精修 → 返回，形成闭环
4. **工具职责分明**：纵向模式无画布工具，精修模式完整工具

### 用户体验目标

- **简单用户**：纵向模式即可完成大部分操作
- **专业用户**：精修模式提供深度编辑能力
- **移动端用户**：简化操作，强制纵向模式

### 后续迭代方向

1. 添加快捷键 Ctrl+L 切换模式
2. 精修模式支持多图对比（可选）
3. 用户偏好持久化（默认模式选择）
4. 精修历史记录（可回溯）
