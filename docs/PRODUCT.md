## 1. 产品是什么

**出图宝 = 中文电商出图与宣传短视频平台**：专注淘宝/京东/抖音等上架场景，先产出可交付的商品图片（主图、套图、详情），再**基于已出图**一键生成对应的产品宣传短视频。

**对外 Slogan**：**商品图到短视频，一套做完上架。**

**品牌名**：出图宝（工程代号 AIMarket，对外统一称出图宝）

### 1.1 Logo 文案规范（域名暂缓）

| 层级 | 文案 / 图形 | 使用场景 |
|------|-------------|----------|
| **图形标** | 叠图卡片 + 右下播放标（橙紫渐变） | 导航、favicon、组件 `BrandMarkIcon` |
| **主标** | 出图宝 | 与图形标组合 |
| **完整组合** | 图形标 + 出图宝 + Slogan 两行 | Hero、登录弹窗 |
| **Slogan** | 商品图到短视频，一套做完上架。 | 与主标同锁 |
| **短副标** | 图到片 · 一套上架 | 极窄位可选 |

**图形标含义**：后方叠层 = 电商套图；前方主卡片 = 商品主图；右下播放圆 = 基于出图的宣传短视频。

**SVG 源文件**：

- `apps/web/public/brand/logo-icon.svg` — 32×32，favicon / 顶栏
- `apps/web/public/brand/logo-mark.svg` — 48×48，详情页 / 物料导出
- `apps/web/src/app/icon.svg` — Next.js 站点图标

**标准锁（Primary lockup）**：

```
[图形标]  出图宝
          商品图到短视频，一套做完上架。
```

**导航锁（Header mark）**：图形标 + 「出图宝」单行。

**禁止**：Logo 区出现 AIMarket、英文副标、单字「出」作图标。

代码组件：`BrandMarkIcon` + `BrandLogo`（`mark` / `lockup` / `icon`）。

---

**定位（精准 ICP）**：**仅服务电商卖家与运营**——不做泛修图、不做娱乐 AIGC、不做非电商设计。

**一句话（对内）**：电商出图工作台 + 出图驱动的宣传短片，不是美图/即梦/灵幻。

**产品边界**：

| 做 | 不做 |
|----|------|
| 电商主图 / 套图 / 详情 / 白底精修 | 人像美颜、证件照、老照片修复（主推） |
| 基于 Project 内商品图生成宣传短视频 | 娱乐短片、剧情视频、社区 feed |
| 出图后的抠图 / 扩图 / 改字等精修 | 泛新媒体封面、非商品类创意 |

| 价值 | 说明 |

## 5. 创作模式产品规则

### 5.1 对话模式

1. 用户输入文本，可选上传 1–4 张图。  
2. 选择模型、张数、分辨率档位。  
3. 提交后显示「生成中」状态。  
4. 完成后 assistant 消息展示图片，可下载。  
5. 继续输入即在同 session 上下文迭代。

**积分**：按 `estimatePoints` 预扣展示，任务成功后结算。

### 5.2 快速模式

- 单轮为主，默认「全能图片 V2」类高速模型。  
- UI 与对话共用面板，减少二次确认。

### 5.3 电商套图 Agent（Phase 2）

| 字段 | 必填 | 说明 |
|------|------|------|
| 产品图 | 是 | 实物/白底图 |
| 参考图 | 否 | 排版风格参考 |
| 品牌名 | 否 | 影响文案与 Logo 区 |
| 平台 | 是 | 淘宝/京东/抖音等 |
| 市场 | 是 | 影响文案语言习惯 |
| 语言 | 是 | 输出文案语言 |
| 产品详情 | 是 | 卖点、规格、受众、材质 |
| 设计师 | 否 | 风格人设 |
| 模型 | 是 | 默认推荐 Pro 档 |

## 6. 灵感库 taxonomy（运营可配置）

| 分类 | 示例场景 |
|------|----------|
| 电商主图 | 白底精修、手持商品、场景主图 |
| 电商套图 | 主图 + 卖点 + 场景 + 详情（Project） |
| 服饰电商 | 虚拟试衣、上身展示 |
| 营销素材 | 促销海报、入驻图（商品向） |
| 宣传短视频 | 基于出图的产品展示短片、旋转/卖点动效 |

每张卡片绑定：**标题、分类、Prompt 模板、变量槽位、模型/比例/分辨率、参考图**。

**做同款流程**：填槽 → 预览 Prompt → 「填入工作台」或 「Fork 为项目」。

## 7. 积分体系（产品规则）

| 项目 | 规则 |
|------|------|
| 货币名称 | 积分（内部代号 credits） |
| 新用户赠送 | 100 积分 |
| 扣费时机 | 任务进入 `running` 时预扣；`failed` 全额退回 |
| 预估因子 | 模型系数 × 张数 × 分辨率系数 |
| 不足提示 | 弹窗引导充值（Phase 3 支付） |

**Phase 1 计价表（示例）**：

| 模型档位 | 单张积分 |
|----------|----------|
| 标准（V2） | 10 |
| Pro | 20 |
| Fast 企业线 | 30 |

分辨率：1K ×1，2K ×1.5，4K ×2。

## 9. 竞品边界与差异化

### 9.1 不正面硬刚

| 竞品 | 主场 | 出图宝 **不做** |
|------|------|-------------------|
| 美图秀秀 | 人像美颜、轻修 | C 端滤镜/美颜流水线 |
| 即梦 | AIGC 娱乐、视频社区 | 纯生成器 + 社区 feed |
| 灵幻 AI | 通用绘图、模型尝鲜 | 无限模型游乐场 |
| 椒图 AI | 中文对话修图 + 电商 | 直接对标，在工作流深度上超越 |

### 9.2 差异化（电商出图 + 宣传短视频）

1. **出图即交付**：Project / 套图 / 平台规格 —— 为上架而生
2. **图生视频闭环**：同一 Project 内商品图 → 宣传短视频，非独立娱乐生成器
3. **同款可复制**：运营模板 + 变量槽 + 品牌资产（后续）
4. **Workspace + 统一 Job**：套图与视频任务可审计、可重跑

### 9.3 能力对照（椒图 AI）

| 能力 | 椒图 | 出图宝 |
|------|------|----------|
| 对话修图 | ✅ | ✅ |
| 灵感同款槽位 | ✅ | ✅ |
| Fork Project | 部分 | ✅ |
| Agent 工具链计划 | 弱 | ✅ `POST /agent/plan` |
| 套图渐进出图 | 弱 | ✅ |
| 多租户 Workspace | 弱 | ✅ |
