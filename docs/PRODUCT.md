## 1. 产品是什么

**AIMarket = 中文商业素材工作流平台**：用「模板同款 + 画布精修 + Agent 编排」产出可交付的工业级图片。

**对外 Slogan**：**中文商业素材，从同款到交付**

**定位（Hybrid ICP）**：电商/新媒体从业者为主，设计师/工作室为辅——赢交付效率，留可控精度。

**一句话（对内）**：会干活的 AI 设计工作台，不是 C 端美颜或娱乐出图器。

## 2. 核心价值主张

| 价值 | 说明 |
|------|------|
| 中文优先 | 无需拼凑英文 Prompt |
| 模板同款 | 灵感槽位填参 + Fork Project，非抄 Prompt |
| 对话式迭代 | 不满意继续说，画布为主视图 |
| 电商开箱即用 | 套图 Agent + 渐进出图 |
| Agent 编排 | 自然语言 → 工具链计划 → 可确认执行 |
| 透明计费 | 生成前可见积分预估 |
| 工作区协作 | 多租户 Workspace（长期壁垒） |

## 4. 页面说明

### 4.1 首页 `/`

**目标**：30 秒内让用户理解产品并尝试第一次输入。

| 区域 | 内容 | 设计要点 |
|------|------|----------|
| 顶栏 | Logo、登录、CTA | 渐变主按钮「免费开始」 |
| 公告条 | 活动信息 | 可关闭，不挡主操作 |
| Hero | 品牌语 + 吉祥物 | 深色 + 右上暖光 |
| 命令面板 | 三 Tab 输入 | 玻璃拟态大圆角容器 |
| 灵感发现 | 场景卡片瀑布流 | 点击打开「做同款」槽位表单；可 Fork 为 Project |

**主文案**：

- 标题：修图，就用 AIMarket  
- 副标题：让每个人都能轻松创作出专业级图片和视频  

### 4.2 工作台 `/studio`

**目标**：完成「上传 → 描述 → 生成 → 下载」闭环。

| 区域 | Phase 1 | Phase 2+ |
|------|---------|----------|
| 顶栏 | 项目名、积分、用户 | 同左 + 套餐入口 |
| 左侧 | 会话历史列表 | + 文件夹 |
| 中央 | 消息时间线 + 结果图 | + 画布对比 |
| 右侧 | 工具图标（占位） | 可用工具面板 |
| 底部 | 创作面板（复用首页） | + @ 引用 |

**空状态文案**：

- 对话：「Hi，我是 AIMarket」+ 引导上传或输入  
- 电商：「套图模式」+ 引导双图上传  

### 4.3 关于 `/about`

## 5. 创作模式产品规则

### 5.1 对话模式

1. 用户输入文本，可选上传 1–4 张图。  
2. 选择模型、张数、分辨率档位。  
3. 提交后显示「生成中」状态。  
4. 完成后 assistant 消息展示图片，可下载。  
5. 继续输入即在同 session 上下文迭代。

**积分**：按 `estimatePoints` 预扣展示，任务成功后结算。

### 5.2 快速模式

- 单轮为主，默认「全能图片 V2」类高速模型。  
- UI 与对话共用面板，减少二次确认。

### 5.3 电商套图 Agent（Phase 2）

| 字段 | 必填 | 说明 |
|------|------|------|
| 产品图 | 是 | 实物/白底图 |
| 参考图 | 否 | 排版风格参考 |
| 品牌名 | 否 | 影响文案与 Logo 区 |
| 平台 | 是 | 淘宝/京东/抖音等 |
| 市场 | 是 | 影响文案语言习惯 |
| 语言 | 是 | 输出文案语言 |
| 产品详情 | 是 | 卖点、规格、受众、材质 |
| 设计师 | 否 | 风格人设 |
| 模型 | 是 | 默认推荐 Pro 档 |

## 6. 灵感库 taxonomy（运营可配置）

| 分类 | 示例场景 |
|------|----------|
| 电商 | 主图、详情图、白底精修、手持商品 |

| 营销 | 海报、卖点图、入驻图 |

| 人像 | 证件照、形象照 |
| 修复 | 老照片上色 |
| 创意 | 影视海报、分镜、超现实 |

每张卡片绑定：**标题、分类、Prompt 模板、变量槽位、模型/比例/分辨率、参考图**。

**做同款流程**：填槽 → 预览 Prompt → 「填入工作台」或 「Fork 为项目」。

## 7. 积分体系（产品规则）

| 项目 | 规则 |
|------|------|
| 货币名称 | 积分（内部代号 credits） |
| 新用户赠送 | 100 积分 |
| 扣费时机 | 任务进入 `running` 时预扣；`failed` 全额退回 |
| 预估因子 | 模型系数 × 张数 × 分辨率系数 |
| 不足提示 | 弹窗引导充值（Phase 3 支付） |

**Phase 1 计价表（示例）**：

| 模型档位 | 单张积分 |
|----------|----------|
| 标准（V2） | 10 |
| Pro | 20 |
| Fast 企业线 | 30 |

分辨率：1K ×1，2K ×1.5，4K ×2。

## 9. 竞品边界与差异化

### 9.1 不正面硬刚

| 竞品 | 主场 | AIMarket **不做** |
|------|------|-------------------|
| 美图秀秀 | 人像美颜、轻修 | C 端滤镜/美颜流水线 |
| 即梦 | AIGC 娱乐、视频社区 | 纯生成器 + 社区 feed |
| 灵幻 AI | 通用绘图、模型尝鲜 | 无限模型游乐场 |
| 椒图 AI | 中文对话修图 + 电商 | 直接对标，在工作流深度上超越 |

### 9.2 差异化三角

1. **交付物导向**：Project / 套图 / 平台规格 / 导出 —— 不是「一张好看的图」
2. **同款可复制**：运营模板 + 变量槽 +（后续）品牌资产库
3. **Workspace + 统一 Job**：可审计、可重跑、可集成 B 端

### 9.3 能力对照（椒图 AI）

| 能力 | 椒图 | AIMarket |
|------|------|----------|
| 对话修图 | ✅ | ✅ |
| 灵感同款槽位 | ✅ | ✅ |
| Fork Project | 部分 | ✅ |
| Agent 工具链计划 | 弱 | ✅ `POST /agent/plan` |
| 套图渐进出图 | 弱 | ✅ |
| 多租户 Workspace | 弱 | ✅ |
