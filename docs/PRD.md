## 1. 背景与目标

### 1.1 背景

国内电商与新媒体对图片素材需求高频，传统 PS 流程门槛高、返工多。竞品 [椒图AI](https://www.jiaotuai.cn/) 验证了「中文对话式修图 + 电商套图 Agent + 多模型路由 + 积分商业化」路径。AIMarket 在同等核心能力基础上，强化可扩展架构与后续 B 端协作能力。

### 1.2 产品愿景

**让每个人都能用中文轻松产出专业级图片与视频素材。**

### 1.3 业务目标（12 个月）

| 指标 | 目标 |
|------|------|
| 注册用户 | 10 万+ |
| 周活跃创作者 | 5,000+ |
| 付费转化率 | ≥ 3% |
| 单次生成满意度（点赞率） | ≥ 70% |

### 1.4 非目标（v1 不做）

- 私有化部署交付
- 完整 PSD 图层编辑
- 自研图像大模型训练
- 小程序 / 原生 App（仅 Web 响应式）


## 4. 三种创作模式（对标规格）

### 4.1 对话模式

- **输入**：自然语言 + 可选多图附件  
- **行为**：多轮上下文；assistant 返回文本 + 图片 URL  
- **占位符示例**：「让照片上的人物笑起来」

### 4.2 快速模式

- **输入**：单轮 prompt，默认更快模型路由  
- **行为**：一次提交一次结果，减少确认步骤  

### 4.3 电商套图 Agent（Phase 2）

- **输入**：产品图、参考排版图、品牌、平台/市场/语言、长文本卖点、设计师人设  
- **输出**：一组详情页/主图/卖点图（≥ 4 张）  
- **默认**：淘宝 / 中国 / 中文

## 5. 非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | 首页 LCP < 2.5s；API P95 < 500ms（不含生成） |
| 可用性 | 核心路径 99.5%（Phase 1 单节点） |
| 安全 | 密码 bcrypt；JWT HS256；上传 MIME 校验 |
| 合规 | 生成内容标注「AI 生成」；用户协议与隐私政策页 |
| 兼容 | Chrome/Safari/Edge 最新两个大版本；移动端适配 |

## 7. 风险与依赖

| 风险 | 缓解 |
|------|------|
| 第三方模型 API 不稳定 | 抽象 Provider + 降级/mock |
